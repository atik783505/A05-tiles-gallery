# 🧱 TilesGallery

**TilesGallery** is a sophisticated, modern, and fully responsive web application built with **Next.js**[cite: 1, 2]. It showcases a curated collection of decorative tiles, emphasizing a premium user interface, seamless user experience, and a scalable frontend architecture[cite: 1, 2].

## 🚀 Deployment Links
*   **Production:** [tiles-gallery.vercel.app](https://a05-tiles-gallery.vercel.app/)


## 🎯 Purpose
The goal of this project is to provide an aesthetic digital catalog for architectural and decorative tiles[cite: 1, 2]. It allows users to explore various textures, designs, and styles through a high-performance, visually engaging interface[cite: 1, 2].

## ✨ Key Features
*   **🖼️ Dynamic Tile Gallery:** Features a beautiful collection of tiles rendered with dynamic data[cite: 1, 2].
*   **🔍 Advanced Search & Filter:** Effortlessly find tiles by name, specific tags, or aesthetic style[cite: 1, 2].
*   **📄 Dynamic Routing:** Includes dedicated detail pages for each tile to provide in-depth information[cite: 1, 2].
*   **📱 Fully Responsive:** Optimized for a flawless experience across mobile, tablet, and desktop devices[cite: 1, 2].
*   **⚡ High Performance:** Leverages Next.js server-side features for fast loading and SEO optimization[cite: 1, 2].
*   **🎨 Modern UI/UX:** Built with Tailwind CSS, featuring glassmorphism effects and clean layouts[cite: 1, 2].
*   **✨ Smooth Animations:** Enhanced with **Animate.css** for a dynamic and engaging user feel[cite: 1].
*   **🔄 Scalable Components:** Utilizes a reusable component architecture for easy maintenance and growth[cite: 2].
